# v1 からの変更点

> v1 詳細: [../現状仕様.md](../現状仕様.md)  
> v2 概要: [概要.md](./概要.md)

---

## 1. 変更マトリクス

| 領域 | v1 | v2 | 変更種別 |
|------|----|----|---------|
| メイン導線 | 決まり字チェック | **初心者モード** | 追加・優先度変更 |
| クイズ形式 | 取り札見て決まり字を思い出す | **4 択（決まり字 → 取り札選択）** | 追加 |
| タイムアタック | 全札一周の所要時間 | **1 分間で得点制 4 択** | 追加 |
| 「設定オン」の意味 | チェック出題対象 | **「覚えた」フラグ**（テスト正解 or 手動でオン） | 意味拡張 |
| 初期状態 | 全札オフ | 全札オフ（継続） | 変更なし |
| 学習 UI | 分類一覧 → 詳細 | 初心者: バッチ順送り / 一覧学習も継続 | 追加 |
| 技術構成 | 素の JS | **Vite + React + TS** | 刷新 |
| PWA 更新 | 手動再インストール | **自動更新フロー** | 改善 |
| インストール案内 | フッター文言のみ | **初回モーダル** | 追加 |
| 効果音 | なし | **正誤 SE** | 追加 |
| デザイン | シンプル Web UI | **ネイティブアプリ風** | 改修 |

## 2. 機能ごとの詳細

### 2.1 継承（そのまま残す）

#### 決まり字チェックモード

v1 の `game-screen` 相当。変更点:

| 項目 | v2 での変更 |
|------|------------|
| 出題対象 | 引き続き「覚えたフラグオン」の札 |
| 逆向き札 | 継続（50% ランダム） |
| タイム履歴 | 直近 3 回を継続 |
| トップからの位置づけ | メイン CTA から **サブメニュー** へ降格 |

#### 一覧学習モード

v1 の `study-list-screen` / `study-detail-screen` 相当。

| 項目 | v2 での変更 |
|------|------------|
| 分類別カード一覧 | 維持 |
| 詳細画面の歌情報 | 維持 |
| フラグトグル | 「覚えたフラグ」として維持（文言調整） |
| トップからの位置づけ | サブメニューへ |

#### 札の設定

v1 の `settings-modal` 相当。

| 項目 | v2 での変更 |
|------|------------|
| 1 文字目グループ一括 | 維持 |
| 個別切替 | 維持 |
| すべてオン / オフ | 維持（オフ時に逆向きもオフ） |
| 逆向き設定 | チェックモード専用として維持 |
| ラベル | 「チェックで使う札」→「覚えた札」等に変更検討 |

#### データ（fudalist）

100 首のデータ構造は **維持**。フィールド追加は v2 初版では不要。

#### PWA 基本

- オフライン対応
- `manifest.json` のスタンドアロン起動
- 全画像プリキャッシュ

### 2.2 新規追加

| 機能 | 説明 |
|------|------|
| 初心者モード | 未覚え 5 首 → 語呂学習 → 4 択テスト |
| 1 分間確認モード | 覚えた札で 60 秒 4 択連打（得点制） |
| 4 択クイズ UI | 共有コンポーネント |
| 効果音 | 正誤フィードバック |
| 進捗表示 | 覚えた首数（ホーム） |
| PWA 初回案内 | インストール手順モーダル |
| PWA 自動更新 | 更新トースト + SW 切替 |

---

### 2.3 廃止・縮小

| v1 の要素 | v2 での扱い |
|----------|------------|
| トップの 3 大ボタン均等配置 | ホームをカード型に再設計。初心者モードが最大 |
| 素の `script.js` 単一ファイル | フレームワーク移行で廃止 |
| 手書き `sw.js` | vite-plugin-pwa に置換 |
| ダブルクリック防止の直書き | フレームワーク / CSS `touch-action` で対応 |

**ユーザー向け機能として廃止するものはなし**（v1 機能はすべてアクセス可能にする方針）。

## 3. 用語の対応表

| v1 UI 文言 | v2 UI 文言（案） |
|-----------|-----------------|
| 設定オン | 覚えた |
| 設定オフ | まだ覚えていない |
| — | 初心者モード学習画面の CTA は「次へ」（「覚えた！」はテスト結果で暗黙的に判定） |
| チェックで使う札 | 覚えた札の管理 |
| 決まり字チェック | 決まり字チェック（そのまま） |
| 決まり字を覚える | 一覧で学ぶ（サブ導線） |

## 4. localStorage の互換性

| v1 キー | v2 での扱い |
|---------|------------|
| `fudanagashi:letters` | **そのまま利用**（覚えたフラグ） |
| `fudanagashi:reverseEnabled` | そのまま利用 |
| `fudanagashi:history` | そのまま利用（チェックモード用） |

v1 ユーザーが v2 にアップデートした場合:

- 既にオンにしていた札 → そのまま「覚えた」として引き継ぎ
- チェック履歴 → 維持
- 追加キーはデフォルト値で初期化

## 7. 移行時のリスクと対策

| リスク | 対策 |
|--------|------|
| PWA キャッシュが v1 のまま | SW 更新フロー + 初回リリース時のリリースノート |
| URL 構造変更でブックマーク切れ | `/` をホームとして維持。旧 URL は不要（SPA 化前と同じルート） |
| ビルド導入で GH Pages 設定変更 | CI 整備・README 更新 |
| 画像パス変更 | `public/` 配下で v1 と同じ相対パスを維持 |
