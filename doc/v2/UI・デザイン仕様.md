# v2 UI・デザイン仕様

> 関連: [概要.md](./概要.md) / [機能仕様.md](./機能仕様.md)

---

## 1. デザインゴール

| ゴール | 説明 |
|--------|------|
| 和風テイストの継承 | v1 の落ち着いた配色・百人一首の雰囲気を残す |
| ネイティブアプリ感 | カード UI・ボトムナビ・余白・タイポグラフィで「Web サイト感」を減らす |
| マルチデバイス | スマホ（〜480px）・タブレット（481〜1024px）・PC（1025px〜）で最適化 |
| 初心者ファースト | 最重要アクションが一目でわかる情報設計 |

---

## 2. デザインコンセプト（案）

### 2.1 キーワード

**「現代の習字教室」** — 和紙・木・緑を基調に、フラットでクリーンなモバイル UI。

### 2.2 v1 からの継承

| 要素 | v1 | v2 |
|------|----|----|
| 背景ベージュ | `#E1DAC3` | 継承 or わずかにトーン調整 |
| アクセント緑 | `#336633` | プライマリカラーとして継承 |
| 決まり字の赤 | `#cc0000` | 継承 |
| 丸ゴシック系フォント | Hiragino Maru Gothic 等 | 見出しに継承。本文はゴシックも併用可 |

### 2.3 v2 で追加する要素

| 要素 | 用途 |
|------|------|
| カードコンポーネント | モード選択・学習進捗・結果表示 |
| セーフエリア対応 | iPhone ノッチ・ホームインジケータ |
| タッチターゲット | 最小 44×44px（Apple HIG 準拠） |
| マイクロアニメーション | 画面遷移・○× 表示・ボタン押下 |
| ダークモード | v2 初版は **スコープ外**（将来候補） |

## 4. タイポグラフィ（案）

| 用途 | サイズ（モバイル基準） | ウェイト |
|------|----------------------|---------|
| アプリタイトル | 28〜32px | Bold |
| 画面タイトル | 22〜24px | Bold |
| 決まり字（クイズ） | 36〜48px | Bold |
| 本文 | 16〜18px | Regular |
| キャプション | 13〜14px | Regular |
| ボタンラベル | 16〜18px | Bold |

### フォントスタック（案）

```css
font-family:
  "Hiragino Maru Gothic ProN",
  "Hiragino Sans",
  "Noto Sans JP",
  sans-serif;
```

Web フォント（Noto Sans JP）の読み込みはパフォーマンスと相談（PWA オフライン時はシステムフォント fallback）。

## 5. レイアウト・ブレークポイント

| 名称 | 幅 | レイアウト方針 |

| mobile | 〜480px | 1 カラム、フル幅ボタン、クイズ 2×2 |

```css

```

## 7. 主要画面ワイヤー（テキスト）

### 7.1 ホーム画面（D-09 確定）

レイアウト: **メイン CTA 2 枚（縦積み）+ 下部 2 カラムサブカード + 設定アイコンボタン**

```
┌─────────────────────────────┐
│  決まり字マスター        ⚙  │  ← 設定は右上アイコンボタン
│   覚えた: 12 / 100 首        │
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │  🌱 初心者モード          │ │  ← 最大・主 CTA
│ │  5首ずつ覚えてテスト      │ │
│ └─────────────────────────┘ │
│ ┌─────────────────────────┐ │
│ │  ⏱ 1分間確認            │ │
│ │  ベスト: 15問            │ │
│ └─────────────────────────┘ │
│ ┌──────────┐ ┌──────────┐  │  ← 2 カラムサブカード
│ │決まり字   │ │一覧で     │  │
│ │チェック   │ │学習       │  │
│ └──────────┘ └──────────┘  │
└─────────────────────────────┘
```

#### ホームの状態表示

| 状態 | 表示 |
|------|------|
| 覚えた 0 首 | 進捗 0%、1 分間モード disabled |
| 覚えた 1〜4 首 | 1 分間「あと N 首で解放」 |
| 覚えた 5 首以上 | 1 分間モード有効 |
| 覚えた 100 首 | 初心者モードは「完走おめでとう」表示 |

### 7.2 初心者モード・学習画面

```
┌─────────────────────────────┐
│ ← トップ          2 / 5      │
├─────────────────────────────┤
│  ┌───────────────────────┐  │
│  │   語呂スライド画像      │  │
│  └───────────────────────┘  │
│         あきの               │
│  秋の田の かりほの庵の...     │
│  1番 / 天智天皇              │
├─────────────────────────────┤
│      [ 次へ → ]              │  ← プライマリ CTA（フラグは変えない）
└─────────────────────────────┘
```

> 覚えたかどうかはテストの正解で判定するため、学習画面に「覚えた！」ボタンは置かない。

### 7.3 4 択クイズ（共有）

```
┌─────────────────────────────┐
│  残り 38秒      正解 5       │  ← 1分間のみ
├─────────────────────────────┤
│                             │
│          はるす              │
│                             │
├─────────────────────────────┤
│   ┌─────┐    ┌─────┐       │
│   │ 札  │    │ 札  │       │
│   └─────┘    └─────┘       │
│   ┌─────┐    ┌─────┐       │
│   │ 札  │    │ 札  │       │
│   └─────┘    └─────┘       │
└─────────────────────────────┘

        [ ○ ] or [ × ]         ← フィードバックオーバーレイ
```

#### クイズのインタラクション

- 札画像タップ → 即フィードバック → **1.0〜1.5 秒待機** → 次問
- タップ領域は画像全体（余白含む）
- 選択後は他選択肢をグレーアウト
- **不正解時**: 正解の取り札を枠・チェック等でハイライト（D-04 確定）

### 7.4 1 分間確認・結果

```
┌─────────────────────────────┐

│      🎉 お疲れさま！         │
│                             │
│     60秒で 14 問正解         │
│     ベスト更新！ 前回 12     │
│                             │
│   [ もう一度 ]  [ トップ ]   │
└─────────────────────────────┘
```

### 7.5 PWA インストール案内

```
┌─────────────────────────────┐
│  📲 ホーム画面に追加しよう    │
│                             │
│  オフラインでも練習できます   │
│  ┌─────────────────────┐   │
│  │ （プラットフォーム別   │   │
│  │   手順イラスト/文言）  │   │

│   [ 後で ]    [ わかった ]   │
└─────────────────────────────┘
```

ボトムシート形式を推奨（モバイルで親指が届きやすい）。

## 9. アニメーション方針

| シーン | 方針 |
|--------|------|
| 画面遷移 | 200〜300ms のフェード or スライド |
| 正解 ○ | スケールイン + フェードアウト（0.8s） |
| 不正解 × | 軽いシェイク + × 表示 |
| ボタン押下 | `scale(0.97)` のタップフィードバック |
| `prefers-reduced-motion` | アニメーション無効化 or 最小化 |

---

## 10. アイコン・イラスト

| 用途 | 方針 |
|------|------|
| モードアイコン | v1 の絵文字（📖🏁⚙️）から SVG アイコンへ移行検討 |
| 空状態 | 「まだ覚えた札がありません」等のシンプルイラスト |
| 効果音 | UI からは見えない。設定でミュート可能に |

---

## 11. アクセシビリティ

| 項目 | 方針 |
|------|------|
| コントラスト | WCAG AA を目標 |
| タップ領域 | 最小 44px |
| `aria-live` | クイズの正誤をスクリーンリーダーに通知 |
| 効果音 | 視覚フィードバックとセット（音なしでも成立） |
| キーボード | PC では 1〜4 キーで選択肢（推奨・余裕があれば） |

---

## 12. フレームワークとの関係

UI 実装は [技術方針.md](./技術方針.md) で選定するフレームワーク上で行う。  
スタイリング方式（CSS Modules / Tailwind / styled-components 等）は技術方針と同時に決定する。
